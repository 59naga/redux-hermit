Redux hermit
---

(WIP) is promise middleware with performing a server-side-rendering after the completion of the promises

Development
---
Requirement global
* NodeJS v5.7.0
* Npm v3.7.1
* Babel-cli v6.5.1 (babel-core v6.5.2)

```bash
git clone https://github.com/59naga/redux-hermit
cd redux-hermit
npm install

npm test
```
